<!-- Backend API contract from Plan 01 -->

GET /api/auth/me response (extended):
```json
{
  "id": "uuid",
  "email": "user@example.com",
  "name": "User Name",
  "locale": "en",
  "has_microsoft": false,
  "registered_via": "local",
  "microsoft_email": null
}
```

GET /api/auth/providers response:
```json
{ "providers": ["local", "microsoft"] }
```

GET /api/auth/bind/microsoft — redirects to Azure OIDC (user must be authenticated)
GET /api/auth/bind/callback — after OIDC, redirects to /account?bound=1

Error codes for bind:
- "sso.already_linked" (409) — Microsoft account already linked to another user
- "auth.sso_not_configured" (501) — OIDC not configured
